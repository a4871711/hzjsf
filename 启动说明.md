# hzjsf 健身房管理系统 - 本地启动说明

## 一、环境要求

| 依赖 | 版本要求 | 说明 |
|------|---------|------|
| JDK | 1.8 | 必须使用 JDK 8，不支持更高版本 |
| Maven | 3.x | 用于构建和启动 |
| MySQL | 5.7+ / 8.x / 9.x | 需创建数据库 `shilijsf` |
| Redis | 任意版本 | 默认连接 127.0.0.1:6379 |

## 二、配置文件修改

### 1. 数据库配置
文件：`src/main/resources/db.properties`

```properties
jdbc.url=jdbc:mysql://127.0.0.1:3306/shilijsf?allowMultiQueries=true&useUnicode=true&characterEncoding=UTF-8&useSSL=false
jdbc.username=root
jdbc.password=你的MySQL密码
```

### 2. Redis 配置
文件：`src/main/resources/config.properties`

```properties
redis.host=127.0.0.1
redis.port=6379
redis.password=           # 本地无密码则留空
redis.database=6
```

### 3. MySQL 驱动版本（如果你的 MySQL 是 8.x 或 9.x）
文件：`pom.xml`，将 mysql-version 改为 8.0.33：

```xml
<mysql-version>8.0.33</mysql-version>
```

原始版本 5.1.38 仅兼容 MySQL 5.x。

## 三、编译问题修复

以下两个文件有无用的 import 会导致编译失败，需删除 `import sun.dc.pr.PRError;` 这一行：

- `src/main/java/com/dlc/modules/api/controller/TeamClassOrderController.java`
- `src/main/java/com/dlc/modules/api/service/impl/IncomePayDetailServiceImpl.java`

## 四、启动方式

### 方式一：Maven Tomcat 插件启动（推荐本地开发）

```bash
# 指定 JDK 8
export JAVA_HOME=/path/to/your/jdk1.8
export PATH=$JAVA_HOME/bin:$PATH

# 启动
mvn tomcat7:run
```

启动成功后访问：
- 后台管理页面：http://localhost:8080/renren-security/login.html
- API 接口：http://localhost:8080/renren-security/api/

### 方式二：打包部署到外部 Tomcat

```bash
mvn clean package -DskipTests
```

将生成的 `target/hzjsf.war` 放入 Tomcat 的 `webapps/` 目录。

## 五、配置文件总览

| 文件 | 用途 |
|------|------|
| `src/main/resources/db.properties` | MySQL 数据库连接 |
| `src/main/resources/config.properties` | Redis、阿里云短信、抖音团购 |
| `src/main/resources/comm.properties` | 项目域名、融云IM、百度人脸识别、文件存储路径 |
| `src/main/resources/spring-jdbc.xml` | Druid 数据源、MyBatis、事务管理 |
| `src/main/resources/spring-redis.xml` | Redis 连接池配置 |
| `src/main/resources/spring-mvc.xml` | Spring MVC、拦截器、定时任务 |
| `src/main/webapp/WEB-INF/web.xml` | Servlet 容器入口配置 |

## 六、项目技术栈

- Spring MVC 4.3.15 + MyBatis-Plus
- Apache Shiro（权限管理）
- FreeMarker（后台页面模板）
- Druid（数据库连接池及监控）
- Jedis（Redis 客户端）
- Quartz（定时任务）

## 七、定时任务

| 任务 | 执行时间 | 说明 |
|------|---------|------|
| updateOrderStatus | 每小时 | 更新卡/商城/私教课/团体课订单状态 |
| deleteFaceRecord | 每天凌晨2点 | 清理过期刷脸数据 |
| updateMemberStatus | 每天凌晨0点 | 会员过期处理 |
| papAutoPay | 每天早上8点 | 微信自动续费代扣 |
